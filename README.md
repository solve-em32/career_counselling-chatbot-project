# 🎓 Career Counselor Bot — Terminal Version

A smart AI-powered career counseling chatbot built in Python.
Helps students figure out career paths, roadmaps, salaries, colleges, and more.

---

## 📁 Folder Structure

```
career_counselor_bot/
│
├── main.py                  ← Run this to start the chatbot
│
├── core/
│   └── engine.py            ← Brain of the bot (logic, intent detection)
│
├── data/
│   ├── careers.json         ← Full career database (10 careers)
│   ├── courses.json         ← Online courses for each career
│   └── aptitude_map.json    ← Interest → Career mapping
│
└── README.md                ← This file
```

---

## 🚀 How to Run

### Step 1: Make sure Python is installed
```bash
python --version
# Should be Python 3.8 or higher
```

### Step 2: Navigate to the project folder
```bash
cd career_counselor_bot
```

### Step 3: Run the bot
```bash
python main.py
```

No external libraries needed! Pure Python only. ✅

## 💬 What You Can Ask

| You Type | Bot Does |
|----------|----------|
| `I want to become a pilot` | Full career overview |
| `How do I become a doctor?` | Step-by-step roadmap |
| `salary of software engineer` | Fresher to senior salary |
| `best colleges for CA` | Top institutes + entrance exams |
| `alternatives to IAS` | Backup career options |
| `courses for data science` | Free online courses |
| `suggest` or `I don't know` | Interest-based assessment |
| `list` | All available careers |
| `exit` or `quit` | Close the bot |
